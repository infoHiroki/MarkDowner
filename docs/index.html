特徴

  • サイドバーインターフェース: 左側でファイル管理、右側で全画面プレビュー
  • 複数ファイル対応: タブで簡単切り替え、×ボタンで閉じる
  • ドラッグ&ドロップ: ファイルを直接ドロップして読み込み
  • ゼロコンフィグ: 設定ファイル不要、すぐに使える
  • 高パフォーマンス: 起動時間 < 100ms、変換速度 < 50ms
