シンプルで高速なマークダウンビューワー

🌐 Web版を試す 💻 ダウンロード 📖 GitHub

📝 即座プレビュー

マークダウンファイルをドラッグ&ドロップするだけで、美しくフォーマットされたプレビューが即座に表示されます。

📁 複数ファイル対応

複数のマークダウンファイルを同時に開き、タブで簡単に切り替えることができます。スクロール位置も個別に記憶されます。

🚀 高速・軽量

Go言語で実装された高速なパーサーと、シングルバイナリでの配布により、どこでも素早く起動できます。

☁️ Web版対応

インストール不要のWeb版をVercelで提供。どこからでもアクセスして、マークダウンファイルを閲覧できます。

🖱️ ドラッグ操作

ファイルのドラッグ&ドロップ読み込みと、タブのドラッグ&ドロップ並び替えに対応しています。

🔧 ゼロコンフィグ

設定ファイル不要で、すぐに使い始められます。ポート番号も環境変数で簡単に変更可能です。
